# DC Algorithm Training

## Fresh Machine Setup

On Windows, run `setup-lab.bat`.

What it does:

- installs Node.js LTS automatically if it is missing
- installs project dependencies
- starts the Vite dev server
- opens the app in your default browser

If you already use `start-app.bat`, it now forwards to the same setup flow.

## Manual Commands

- `npm run dev`
- `npm run build`
- `npm run lint`
